# amplify

## Requirements
```
sudo npm install -g @aws-amplify/cli
npx
```

## Install
```
amplify init         
amplify add api
pip3 install amplify
pip3 install --user pipenv
pipenv install
```

## Edit
Edit lambda on <a href=amplify/backend/function/amplify/src/index.py>index.py</a>

## Test
amplify mock api

## Run
```
amplify push
amplify publish
```
